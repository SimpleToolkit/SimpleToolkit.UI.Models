<h3 align="center"><b>Set of helpers, class extensions, UI controls used in my other C# projects</b></h3>

<p align="center">SimpleToolkit is package of useful classes, helpers, extensions and UI controls, I use in my C# projects. UI.Models subpackage contains models and helper classes for creating an UI.</p>

- **NavigationViewItem** - model which can be used to implement NavigationViewService-less navigation in MVVM

## Installation and usage

You can download package from official NuGet registry or .nupkg file itself from Releases tab.

**CLI:**

```
dotnet add package SimpleToolkit.UI.Models
```

**Package reference in .csproj file:**

```
<PackageReference Include="SimpleToolkit.UI.Models" Version="<version>" />
```
